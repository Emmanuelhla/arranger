How to Play

1. Select a puzzle category from the main screen.
2. Drag and drop the scrambled pieces from the right panel into the correct positions on the left puzzle board.
3. Complete all the puzzles in a category to win!
4. Use the "Show Hint" button if you get stuck.
5. Use the "Reset Puzzle" button to reshuffle the current puzzle.
6. Click "Exit" to go back to the category selection.
